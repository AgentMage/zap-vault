# Zap Oracle Vault

An offline Obsidian vault for the [Zap Oracle](https://zaporacle.com) — a free 720-card divination system created by Jonathan Zap.

## What is the Zap Oracle?

The Zap Oracle is a tool for self-reflection and personal divination. Each card carries a title, an image, and extended philosophical text drawn from Jungian psychology, mythology, and Jonathan Zap's own writing on consciousness and human transformation. You draw a card, sit with what it surfaces, and write your own response to it.

Jonathan Zap is a philosopher, author, and teacher based in Boulder, Colorado, with a background in Jungian psychology, archetypes, dream interpretation, and esoteric philosophy. The full oracle is available free at [zaporacle.com](https://zaporacle.com).

This vault contains 557 unique cards (duplicates and disabled cards removed from the source database).

## Install

1. Install [Obsidian](https://obsidian.md) on your device
2. Copy the `zap-oracle-vault/` folder to your device
3. Open Obsidian → **Open folder as vault** → select `zap-oracle-vault/`
4. When prompted about community plugins, click **Trust and Enable**

That's it. Open `Oracle.md` to draw your first card.

## Usage

- **Draw** — open `Oracle.md` and tap the Draw link
- **Notes** — each card has a Notes section at the bottom for your own writing
- **Draw Log** — add a line to the Draw Log in `Oracle.md` after each draw: `- 2026-06-15 [[card-name]]`

## Structure

```
zap-oracle-vault/
  Oracle.md          ← start here
  cards/             ← 557 card notes
  images/            ← card images (works offline)
  .obsidian/         ← Obsidian config + Dataview plugin
```
